# 90-Day Polymath Roadmap

A web application for tracking a 90-day journey across five disciplines: Science, Arts, History, Math & Logic, and Physical Skills.

## Overview

This project transforms a single HTML file into a structured web application with separate HTML, CSS, and JavaScript files. It provides an interactive roadmap for developing polymath skills through daily practice and weekly projects.

## Features

- **Overview**: Three-phase structure (Foundation, Connection, Synthesis)
- **Daily Schedule**: Sample weekday routine with time blocks
- **Resources**: Curated free learning resources for each discipline
- **Weekly Projects**: Synthesis projects combining multiple disciplines
- **Progress Tracking**: SMART goals, progress bars, and milestone checklists

## File Structure

```
polymath/
├── index.html      # Main HTML structure
├── styles.css      # CSS styles and layout
├── script.js       # JavaScript functionality
└── README.md       # This file
```

## Usage

1. Open `index.html` in a web browser.
2. Navigate through the tabs to explore different sections.
3. Click on resource links to open them in new tabs.
4. Use the progress tab to track milestones and goals.

## Browser Compatibility

Works in modern web browsers that support ES6 JavaScript and CSS Grid/Flexbox.

## Customization

- Edit `script.js` to modify schedules, resources, or projects.
- Update `styles.css` for visual changes.
- Adjust `index.html` for structural modifications.
